This package defines the classes and data structures required to work with j-Interop. It also implements the core principles of DCOM specifications . A good starting point is to get familiar with {@see #JISession} and {@see #JIComServer} classes. They together serve as an entry point into the COM world. A quick glance at {@see #IJIComObject} would familiarize the user with basic COM APIs. The event call back mechanism is implemented by {@see #JIJavaCoClass } , {@see #JIInterfaceDefinition } , {@see #JIMethodDescriptor } and {@see #JIParameterObject } classes.
